---Recipe Searching---

   Description: 
Our project can search for recipes using area(chinese, canadian, japanese,..), name, or the first letter in its name. Moreover, it can show ingredients needed and detailed instructions to make it. There is a button at the bottom for each recipe that links to a video where you can watch how to make the meal step by step.
